# 基于Javaweb大学生兼职平台系统

#### 介绍
该项目是一个大学生校园兼职平台，主要用户是商家、学生、管理员。商家可以发布兼职信息，发布后，管理员进入后台确认信息是否有效，并且审核该兼职信息。学生登录平台后，可以查看兼职信息，并且，可以在线和商家聊天，聊天的内容持久化到mysql数据库中。
该网站还有论坛交流的功能，用户可以在论坛发表贴子，并且，可以在帖子下面回复。论坛版块设置有3块广告区，管理员可以在后台对广告进行增删改查。
